# 🤖 LINGO BOT - WhatsApp Session Generator

A powerful WhatsApp session generator supporting both **Pairing Code** and **QR Code** authentication methods.

## ✨ Features

- 🔗 **Pairing Code Generation** - Get 8-digit pairing code with phone number
- 📱 **QR Code Scanner** - Traditional QR code login method
- ⚡ **Fast & Reliable** - Optimized Baileys engine
- 🔒 **Secure** - Sessions automatically deleted after delivery
- 🌐 **Web Interface** - Beautiful, responsive UI
- 🚀 **Easy Deployment** - One-click Heroku deployment

## 🛠️ Tech Stack

- **Backend**: Node.js, Express, Baileys (WhatsApp Web API)
- **Frontend**: HTML5, CSS3, JavaScript
- **Database**: File-based session storage (auto-deleted)
- **Process Manager**: PM2
- **Deployment**: Heroku, VPS, cPanel
